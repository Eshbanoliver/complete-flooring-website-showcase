# Complete Flooring Company - Premium Showcase Website

A high-end, modern, and interactive web application designed for a premium flooring solutions provider. This showcase website emphasizes visual excellence through glassmorphism, fluid animations, and a rich, curated design system.

## 🚀 Tech Stack

- **Core**: React 19 (Vite)
- **Styling**: Vanilla CSS (Modern Design System)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router Dom 7
- **Development**: Vite 8 (Ultra-fast HMR)

## 🎨 Design Philosophy & Creativity

The website is built with a **Premium & State-of-the-Art** aesthetic, focusing on:
- **Glassmorphism**: Subtle backdrop blurs and semi-transparent "glass cards" used across the UI for a clean, sophisticated atmosphere.
- **Dynamic Micro-Animations**: Interactive hover effects (magnetic-like interactions, 3D tilts) and staggered entry animations for a living, breathing user experience.
- **Visual Depth**: Use of decorative background blobs, custom gradients, and soft glowing rings around interactive elements.
- **Interactive Forms**: A modern contact experience with floating focus lines, animated icon overlays, and "button-shine" effects.

## 🌈 Theme & Color System

The design uses a carefully curated color palette to evoke luxury and professionalism:
- **Primary Color (`#3A0353`)**: Deep Royal Purple for branding and trust.
- **Secondary Color (`#F8D299`)**: Soft Golden Sand for highlights and elegance.
- **Accent Color (`#F59E51`)**: Vibrant Gold/Orange for high-impact call-to-actions.
- **Micro-interactions**: Use of these colors in gradients and glowing rings (`box-shadow` glows).

## ✨ Key Features Implemented

- **Animated Hero Section**: A multi-slide background experience with fluid text transitions.
- **Dynamic Contact Experience**: 
  - Interactive Google Maps integration with a floating office detail overlay.
  - Advanced form validation and visual feedback using Framer Motion.
- **Service & Product Showcases**: High-impact cards with image zoom and glassmorphism overlays.
- **Testimonial Slider**: An interactive slider featuring client success stories with an auto-play mechanism.
- **Core Values & Mission**: Clean, structured sections with custom icons and staggered reveals.
- **Full Responsiveness**: Custom media queries ensuring a high-end experience from 4K displays down to iPhone SE.
- **Site Identity**: Custom branding, favicon integration, and deep SEO meta-tagging (Open Graph & Twitter Cards).

## 🛠️ Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Build for Production
```bash
npm run build
```
